background-image: url(Resource/img_01.jpg);

默认，背景图片小于元素，会平铺。图片大小大于元素，图片会显示不完全。

background-repeat: no-repeat; 背景图片不重复

repeat-x，背景图水平方向重复，导航条可以使用一条图片水平平铺显示。

background-position: right bottom; 设置图片的位置

background-position: 20px 20px;

background-attachment: fixed; scroll 是否跟随滚动。

fixed位置时相对于浏览器的窗口, 所以一般只设置给body元素，不设置给其他元素

background:可以同时设置所有背景相关的样式，没有顺序、数量要求，不写的就使用默认值
