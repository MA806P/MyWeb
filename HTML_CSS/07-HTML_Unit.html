长度单位 px-像素。可以使用个百分比，父元素长度改变子元素也会变。

em是响度当前元素字体大小来计算的 1em=1font-size

颜色 background-color: #ff0000; rgb(0, 255, 100); rgb(0, 255, 100); rgb(0%, 80%, 100%);
#abc = #aabbcc

color 字体颜色，font-size 字体大小
font-family 同时采用多个字体时，优先使用前一个，没有在下一个

字体分类 5大类
serif 衬线字体
sans-serif 非衬线字体
monospace 等宽字体
cursive 草书字体
fantasy 魔幻字体

一般将字体大类放在font-family的最后一个，要是前面的字体都找不到，这个兜底的

font-style: italic; 文字斜体。可选 normal oblique ..

font-weight: bold;设置加粗效果 100-900之间9个值。系统可能没有，达不到想要效果

字体样式设置在一起 abcdefgHIJKL，字体必须是最后一个 字体大小是倒数第二个

line-height:40px; 行高。可以写 100% 相对于 font-size 来计算的。

行高也可在 font 中设置，font:20px/30px; 字体20行高30

若font没写行高会用默认的行高，line-height要写在font之后才起作用

text-transform 设置文本大小写。text-align 文本对齐。text-indent 文本缩进 正负值
