内联元素的边距，和块元素的边距设置不一样

内联元素不能设置 width height。可以设置水平方向的边距 padding-left，垂直方向边距不会影响页面布局

内联元素 margin 只支持水平方向，不支持垂直方向

  • 可以通过 display 样式可以修改元素的类型
  • inline作为内联样式显示
  • block作为块元素显示
  • inline-block 既有行内元素的特点又有块元素的特点。既可设置宽高又不会独占一行
  • none 不显示元素，不占据页面的位置
  • visibility:hidden 不显示，但是还占据位置
