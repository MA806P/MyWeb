文档流处在网页的最底层，表示的是一个页面中的位置，所创建的元素都在文档流中。
当元素宽高为auto时，指定内边距不会影响可见框的大小，会自动改变宽，适应内边距。

块元素
会独占一行，自上向下排列
默认值宽度父元素的100%
默认高度被内容撑开
内联元素
只占自身的大小，默认从左向右排列，一行不够换到下一行
在文档流中，默认宽高都会被内容撑开

想要使div水平排列，可以使块元素脱离文档流，使用 float: 使元素浮动

none默认值在文档流中排列。 left脱离文档流，想页面左侧浮动。right脱离文档流，想右侧浮动

当一个元素浮动脱离文档流后，下面的元素会向上移动。如果浮动元素上面试一个没有浮动的块元素，则浮动不会超过块元素

浮动的元素不会超过上面的兄弟元素，最多一边齐

浮动元素不会盖住文字，文字会环绕在浮动元素的周围

this is div 当元素设置为浮动，会完全脱离文档流，块元素的宽度和高度会被内容撑开。
this is span 内联元素脱离文档流会变成块元素，设置宽高起作用了。
