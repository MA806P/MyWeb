js 严格区分大小写，每条语句以;分号结尾，如果不写分号，浏览器会自动添加，但是会消耗资源，浏览器可能会错加分号，所以必须写。

js 会忽略空格和换行，可以对代码进行格式化。

js 中所有的可以由我们命名的都可以成为是标识符，变量名、函数名、属性名
标识符不能以数字开头，不能是ES中的关键字或保留字
js 保存标识符采用Unicode编码，理论上所有的utf-8含有的内容都可以作为标识符
js 中有六种数据类型
String Number Boolean Null Undefined(基本数据类型) Object(引用数据类型)
String
String 单引号 双引号都行，但是不要混着用，引号不能嵌套 双引号里不能用双引号 单引号里也不能使用单引号，可用\转义
var a = "he say:\"hello a \t bc\" "
Number
console.log(Number.MAX_VALUE); 超过了最大值返回一个 Infinity 表示无穷 也是Number类型的
var a = NaN; 也是Number类型的
Null
Null 类型的值只有一个，就是null，用来表示一个为空的对象。
var a = null; typeof a = object
Undefined 类型值就一个 undefined
类型转换
.toString() 转换成 String， null undefined 不能用 toString 方法，使用 String(null)
parseInt('12a34') = 12;parseInt('12.45a34') = 12;
Number('123') = 123; Number('12a') = NaN;
Number(' ') = 0; Number(false) = 0; Number(undefined) = NaN; Number(null) = 0;
Boolean(123) = true; Boolean('abc')=true, Boolean(' ')=true, Boolean('')=false, Boolean(undefined)=false, Boolean(null)=false
进制
16进制 0x 开头； 8进制 0 开头；
var a = 070; parseInt(a, 10) = 56; 第二个参数以10机制转换，8进制转换成10进制
运算符
'abc' + 1 = 'abc1'; 123 + "1" = '1231'; 1+''='1';
NaN+任何值=NaN; true + 1 = 2, true + "123" = "true123", null + 1 = 1,
1 + 2 + '3' = '33'; '1'+2+3='123';
100 - '1' = 99;
2 * '2'=4, 2 * null=0, 2 * undefined=NaN, 2 * NaN=NaN
可以 -0 *1 /1 进行转换为 Number
自增自减
a++ ++a, 变量a的值都会立即+1，a++的值=a， ++a的值 = a+1
var d=20; var r = d++ + ++d + d; r=20+22+22=64, d=22
var d=20; d = d++; d是20
逻辑运算
var a = 22; !!a = true; 这种方法可以隐式转换成Boolean类型。!!'abc' = true
'1' < 5 true, '4' > '2' true, 两个都是字符串比较，不会将其转换为数字，会分别比较字符串中的Unicode编码

相等运算符
== 如果比较的值类型不同，会自动进行类型转换，转换成相同的类型。
NaN 不和任何值相等，包括它本身 isNaN()函数检查值是否NaN
'1' == 1 true, true == '1' true, null == false false,
null == null true, NaN == NaN false, null == undefined true, undefined == undefined true
=== 全等，不会做类型转换，null === undefined false, undefined === undefined true, null===null true,
